Pygame Jump King

I recreated the game Jump King in Pygame. Just like Jump King, you use left and right arrows to move and space bar to charge up a jump. I used Code Bullets Github repository for the background images/sprites. I learned to implement collision detection as well as map making to recreate the Jump King map layout.
